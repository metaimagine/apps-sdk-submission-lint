# Changelog

All notable changes to `apps-sdk-submission-lint` will be documented in this file.

## v0.1.0 - 2026-05-03

Initial launch-ready release.

- Added offline JSON manifest scanning for common Apps SDK submission metadata gaps.
- Added checks for app naming, descriptions, privacy policy URLs, CSP domains, tool annotation hints, screenshots, and unsafe approval or certification wording.
- Added text and JSON CLI output for local review and CI automation.
- Added valid and invalid example manifests, tests, license, CI workflow, contribution guide, demo asset, and launch copy.

Disclaimer: this project is an independent lint helper. It is not OpenAI approval, certification, endorsement, or a guarantee that an app will pass any submission review.
